PoC 03 — Picker / Camera Overlay Navigation

Re-test of CVE-2025-43228. Open a file/camera picker, then trigger a navigation after a delay. Oracle = does the top page navigate (or a popup open showing a DIFFERENT origin) WHILE the picker/camera is still on screen? If yes → origin-confusion overlay bypass. Best on 3rd-party iOS browsers (Chrome/Edge/Firefox) and in standalone PWA mode.

To test PWA standalone mode: add this page to Home Screen, open it there (no URL bar), then run. No URL bar = no origin signal, so any wrong-origin navigation is more severe.
